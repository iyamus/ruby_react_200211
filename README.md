** Install rails and  react **
```
	• rails new rails-react-tutorial --webpack=react
```
** Add webpacker and react-rails **
```
  • gem 'webpacker', '~> 4.0'
	• gem 'react-rails'
```
** Install bundles **
```
	•  cd project folder
	•  bundle install
	•  rails webpacker:install:react
	•  rails generate react:install
```
** run server **
```
•  rails s
```
